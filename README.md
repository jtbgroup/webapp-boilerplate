This project is a dummy one create to have a base structure before starting the development of a web application. 

Things to do to use this boilerplate:

* clone de repo; don't fork it as it's not the intent at all of this project
* disconnect from the original git repo
* push on a new git repo
* Replace all the **webappboilerplate** occurences (package names, application name, databases, ...) by your own project name
* change the ports according to your needs in the docker-compose files
* run make `dev-start`
